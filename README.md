# Wuwuwu Sleight

## How do I install these formulae?

`brew install wuwuwu/sleight/<formula>`

Or `brew tap wuwuwu/sleight` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "wuwuwu/sleight"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
